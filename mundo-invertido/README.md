# Jornada ao Mundo Invertido

Uma experiência interativa inspirada no universo de Stranger Things. O objetivo? Enfrentar os desafios do Mundo Invertido e salvar Hawkings das mãos de Vecna. Prepare-se para navegar por uma interface envolvente e cheia de mistérios!

## Tecnologias Utilizadas
- HTML: estrutura base das páginas
- CSS: estilização, animações, temas escuro e claro
- JavaScript: interações dinâmicas e integração com serviços

## Itens abordados e que o projeto possuí
- HTML
  - Estrutura semântica
  - Acessibilidade e SEO
  - Integração de formulário com o Google Forms
- CSS
  - Flexbox
  - Pseudo-elementos e pseudo-classes
  - Temas claros/escuros com variáveis
  - Animações visuais
- JavaScript
  - Manipulação do DOM
  - Execução de eventos
### Temas Personalizados
- O projeto conta com dois estilos visuais:
  - Modo Claro: design suave e fontes legíveis
  - Modo Escuro: visual imersivo com fonte glitch de clima misterioso
  - As cores, imagens e fontes são gerenciadas via variáveis CSS, permitindo mudanças rápidas e organizadas.
